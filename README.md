### Programming-Theory repository
This repo has mainly one goal only, which is to apply the concept of object-oriented programming (OOP) while also using a version control tool, as instructed in the Jr. programmer pathway.
Four components of OOP will be applied:
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation

##
A MVP is not required. Should take a week to implement everything. \
**Update:** This mini project has successfully concluded on Apr 19th, 2026, despite achieving things less than what were initially planned.
